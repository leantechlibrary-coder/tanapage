# 棚主ガイド（管理ページの使い方）

あなたの棚には2つのURLがあります。運営者から受け取ってください（自分でセットアップした人は自分で決めたものです）:

- **棚ページ** `…/?s=（棚ID）` — お客さんが見るページ。QRコードで棚に貼る
- **管理ページ** `…/admin.html?s=（棚ID）` — このガイドで説明するページ

スマホだけで完結します。ブックマークしておくと楽です。

## はじめての日: 棚キーを決める

管理ページを初めて開くと「棚キーの設定」画面になります。8文字以上の合言葉を決めてください。

- 棚キーはパスワードと同じ扱いのもの。パスワード管理アプリ（iCloud キーチェーン等）での生成・保存がおすすめ
- キーは暗号学的ハッシュでしか保存されません。**忘れると誰にも分からない**ので、忘れたら運営者にリセットを頼んでください（リセット後、同じ画面で決め直せます）

2回目からは、棚ID＋棚キーでログインします。

## 購入報告を承認する

お客さんが「買いました」を送ると、管理ページの**購入報告**に並びます。

1. **棚の現物がなくなっていることを確かめてから**「承認して販売済にする」を押す
2. 承認すると本は販売済になり、棚ページから消え、販売履歴に記録される
3. いたずらや間違いなら「却下」— 本は棚に残る

「URLを含むため保留」と付いた報告は、宣伝スパムの可能性があるので一言の内容を見て判断してください。

誤って承認したときは、在庫の一覧でその本の「**棚に戻す**」を押せば元に戻ります。

## 在庫を管理する

- **＋ 本を追加** — 書名（必須）・著者・価格・棚主の一言・返礼メッセージ
- **編集** — 同じ項目をあとから直せる
- **撤去** — 棚ページと一覧から消える（記録は残る）

**返礼メッセージ**は、その本を買った人にだけ表示される隠しことばです。空にしておくと、共通の返礼（下記）からランダムで1つ表示されます。

## 棚のことば

- **棚主より** — 棚ページに公開されるメッセージ。「先週◯◯を買ってくださった方へ」のような、ゆるい返信にも使えます。空なら欄ごと非表示
- **共通の返礼メッセージ** — 1行に1つ。購入報告のあとにランダムで1件だけ表示されます

## 販売履歴

月別の冊数・売上と累計が見られます。精算のメモにどうぞ。

## こまったとき

| 症状 | 対処 |
|---|---|
| 棚キーを忘れた | 運営者に連絡してリセットしてもらう → 開き直すと再設定画面になる |
| 「棚キーが違います」 | 棚IDと棚キーの組を確認。棚IDはURLの `?s=` の後ろの文字列 |
| 「GAS側が古いままです」と出る | 運営者に連絡（バックエンドの更新忘れ） |
| 通信できない | 電波のよいところでもう一度。書きかけの文章はコピーしてから再読み込みを |
